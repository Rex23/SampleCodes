#Aim of the package:
The package is a matching engine and in-memory database that takes order requests and matches them against previous orders to produce trades.

The Makefile is under ‘./Linux’. To build the package, on the Linux terminal, type:
make
To run the executable, type:
./Matching_Engine
To run the 20 testing cases before calling the matching engine, type:
./Matching_Engine RUN_TESTS
